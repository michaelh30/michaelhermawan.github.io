# Michael Hermawan Portfolio

Personal portfolio website built with plain HTML and CSS for GitHub Pages.

## Overview

This website includes:

- Responsive single-page portfolio layout
- Cyan, white, and blue theme using `#4A90E2` and `#1E3A8A`
- Hero section with placeholder profile image
- About, skills, portfolio, education, experience, and contact sections
- Mobile-friendly navigation

## Files

- `index.html` - Main website structure and portfolio content
- `styles.css` - Styling, layout, colors, and responsive design
- `.gitignore` - Ignored local, editor, dependency, and log files

## Customization

Replace the placeholder text, contact links, education, experience, and project details in `index.html`.

To change images, replace the dummy `placehold.co` image URLs with your own image paths or hosted image URLs.

## Deployment

This repository is ready for GitHub Pages. Push the files to the `main` branch, then enable GitHub Pages from the repository settings if it is not already active.
